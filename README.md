# pygame-pequenos-genios

Colección de juegos desarrollados con Pygame como parte del libro **"Python para Pequeños Genios: Da Vida a tus Videojuegos con Gráficos"**.

## 🎮 Descarga rápida

Cada capítulo incluye un juego listo para ejecutar (o varios en algunos casos).  
Puedes descargar directamente los ejecutables desde la sección de **Releases**:

👉 https://github.com/crb777/pygame-pequenos-genios/releases

## 📦 Contenido

- Juegos completos desarrollados paso a paso
- Ejecutables (.exe) listos para usar
- Recursos (imágenes y sonidos)

## 🧠 Objetivo

Aprender programación en Python creando juegos sencillos, visuales y divertidos mediante pygame.

## ▶️ Ejecución

Puedes:
- Ejecutar directamente el `.exe`
- O abrir el código en Python y modificarlo

## 📁 Estructura

Cada capítulo incluye:
- Código del juego
- Recursos necesarios
- Ejecutable listo para jugar
